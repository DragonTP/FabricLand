# 🪄 FabricLand College Explorer Power BI Dashboard

## 📖 Overview
This project is a **Power BI dashboard** developed for the **Microsoft Fabric Student Contest**.

The dashboard helps students evaluate the **Return on Investment (ROI)** of different colleges by transforming complex academic data into an **interactive decision-making tool**.

The report is designed with a **wizard-themed interface** and includes multiple pages that guide users through college performance, program quality, and personalized recommendations.

![Overview image of Fabricland report](https://github.com/DragonTP/FabricLand/blob/bdbabef56b46a324fd54664ef8eed2fd0949f643/fabricland%20project%20image.jpg)

---

## 🛠️ Tech Stack

**Power BI**

Used for the entire workflow:

- Data processing  
- Advanced **DAX calculations** for ROI scoring  
- Interactive dashboard design  

---

## ✨ Key Features

### 🏫 Page 1: College Overview

Provides a **high-level summary of college performance**.

Key metrics include:

- **Tuition**
- **Salary**
- **Happiness Score**
- **Placement Rate**
- **Magic Score**
- **Total Programs**

Visualizations include:

- **Magic Score ranking** of top-performing colleges
- **Salary vs Tuition scatter plot** to evaluate financial ROI
- **Tier distribution** of schools (Tier A, B, C)
- **Net benefit trend** combining affordability and salary outcomes
- **College performance ranking table**

---

### 📖 Page 2: Comparing Program Quality Across Majors & Levels

Analyzes **program distribution and academic outcomes across majors and education levels**.

Features include:

- **Students by Major & Level**  
  Shows enrollment patterns across **Undergraduate, Graduate, and Apprenticeship programs**.

- **Programs Distribution**  
  A **pie chart visualization** displaying the distribution of majors across the education system.

- **Popularity Trends Over Time**  
  Tracks how the popularity of different **majors changes over time**.

- **Salary & Faculty by Level**  
  Compares **graduate salary outcomes and faculty ratings** across education levels.

- **Outcome Score by Major**  
  A combined metric that highlights **majors offering the best overall value** based on career outcomes.

---

### 📘 Page 3: Program Detail Analysis

Provides a **deep dive into individual college programs**.

Key features:

- **Drill-through functionality** from the overview page
- Ability to analyze **specific academic tracks**
- Insights into:
  - Student growth trends
  - Program difficulty
  - Graduate salary outcomes

---

### 🎓 Page 4: Personalized College Matching

Helps users identify the **best-fit college based on personal priorities**.

Features include:

- **Customizable scoring weights** allowing users to prioritize:
  - Career Outcomes
  - Affordability
  - Student Experience

- **Key Influencers visualization** showing factors that most impact college quality
- **Decomposition Tree** analyzing drivers behind starting salaries across majors
